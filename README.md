# PDF to Audio Converter

A web application that converts PDF documents into natural-sounding audio using AI-powered text-to-speech. Built with React, FastAPI, Edge-TTS, and PyMuPDF.

## Features

- PDF upload
- AI voice selection
- MP3 generation
- Audio preview
- Download MP3
- Responsive UI

## Tech Stack

React • Vite • FastAPI • Edge-TTS • PyMuPDF • Tailwind CSS

## Run Locally

```bash
# Backend
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

# Frontend
cd frontend
npm install
npm run dev
```
